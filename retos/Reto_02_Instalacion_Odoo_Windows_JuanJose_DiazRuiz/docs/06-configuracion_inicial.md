# 06 — Configuración inicial

1. Localiza archivos de **configuración** (si aplica) y el puerto por defecto (ej. `8069`).
Para hacer estas configuraciones lo puedes hacer directamente desde el instalador, aunque personalmente recomiendo dejar el puerto ocupado por el que está predeterminado, pues no es usado por ningún otro servivio local común. También puedes acceder directamente desde la applicación: 
![](../assets/img/06-configuracion_inicial/ajustes.png)
2. Anota la **URL de acceso local** (p. ej., `http://localhost:8069`).
En este caso he dejado la URL de acceso en el valor predeterminado.
> Resultado esperado: parámetros básicos identificados/documentados.
